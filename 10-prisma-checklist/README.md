# PRISMA 2020 Compliance (SRBIA Review)

This folder is the central location for everything related to PRISMA 2020 compliance for our Systematic Literature Review on Scalable Rule-Based Inference Architecture.

## Files
- `prisma-2020-checklist.xlsx` — Tracking sheet for all 27 PRISMA items.
- `prisma-2020-statement-citation.txt` — Official citation information.
- This README

The final, page-mapped version for the manuscript is also available at `../08-manuscript/prisma-2020-checklist-completed.csv`.

## Why PRISMA 2020?

For this KBS project, we adopted PRISMA 2020 because:
1. It is the gold standard for high-impact journals like **Knowledge-Based Systems (Elsevier)**.
2. It provides a more modern framework (2020) compared to Kitchenham (2007).
3. It ensures 100% transparency in reporting how we went from 500 records to 12 final studies.

## How we used the Checklist
- **Step 1:** Used it during the search string design (Folder 01).
- **Step 2:** Followed Item 11 for Risk of Bias assessment (Folder 06).
- **Step 3:** Mapped all 27 items to our final project report.
